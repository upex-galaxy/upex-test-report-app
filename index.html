UPEX Test Report Portal

Visualize and manage your Allure test reports with ease

UPEX Test Report Portal is a modern web application designed to help QA teams and developers visualize, organize, and analyze Allure test reports across different environments and test strategies.

Built with Next.js and featuring a sleek, space-themed UI, this portal provides a centralized location for all your test reporting needs.

View on GitHub

Automatic Detection

Automatically detects and organizes Allure reports based on environment, strategy, and build number.

Secure Access

Protected with NextAuth.js, supporting multiple authentication methods including GitHub, Google, and credentials.

Modern UI

Sleek, responsive interface with a space-themed design featuring neon accents and intuitive navigation.

Getting Started

Clone the repository, configure your environment variables, and deploy to Vercel for a complete test reporting solution. Check the documentation for detailed setup instructions.
